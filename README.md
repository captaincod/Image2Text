# Символы в текст  
Программа: `image2text.py`  
Картинки, на которых распознавались символы: `out/{number}.png`  
Тренировочный датасет: `out/train`  

Вывод программы:  
```
- in To o rTTnTT
nrL in yoo TyTIT
oynhon inIOTTITInr
Rrnn innAFE
Lro in Lonn
Iononnn ion in I-Tn
```
